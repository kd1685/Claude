# Setup guides

Pick the path that matches what you want right now:

| Guide | Use it when |
|-------|-------------|
| **[1-QUICKSTART.md](1-QUICKSTART.md)** | You just want to see it running on your own computer with demo data (no game, ~2 min). |
| **[2-VPS-SETUP.md](2-VPS-SETUP.md)** | You want it live on a server, on the internet, with login (the real deployment). |
| **[3-CONNECT-GAME.md](3-CONNECT-GAME.md)** | You're ready to point it at a real Rise of Kingdoms client and calibrate it. |
| **[4-DAILY-USE.md](4-DAILY-USE.md)** | It's set up and you want the day-to-day "how do I scan / give titles / run a rotation" cheat sheet. |
| **[5-GITHUB-PAGES.md](5-GITHUB-PAGES.md)** | You want the website public for free on GitHub Pages, auto-updating once the backend is online. |

Recommended order: **1 → 2 → 3**, then keep **4** handy.
