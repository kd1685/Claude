# 4. Daily use cheat sheet

Quick reference for officers once everything is set up.

### Logging in
- Public pages (Dashboard, Power/KP, Dead, DKP, Rallies, Governors, Map) need **no login**.
- The **Control** and **Audit** pages need your officer login.
- First login with a temp password → you'll be asked to set your own.
- Change your own password any time: Control → **My account**.

### Update the data (scan)
Control → **Scan rankings** → choose Power / Kill Points / Dead → **Start**.
Watch the **Command queue**; the data pages update when it's done. (Or let the
**Automatic scans** do it daily — times are UTC.)

### Give one title
Control → **Grant title** → pick governor + title → **Give title**.

### Title rotation (title duty for many people)
Control → **Title rotation**:
1. Pick the **title** and **minutes each** (default 3).
2. **Add** each governor — they run in the order you add them.
3. **Start rotation.**
The live view shows who's holding it now and the countdown. Use **Skip current**
to move on early, or **Cancel rotation** to stop.
*While a rotation runs, other control actions wait until it finishes.*

### Change a rank
Control → **Change rank** → pick governor + rank. Any officer can set R1–R4;
**only admins can promote to R5.**

### Find someone on the map
Control → **Locate on map** → pick governor. The result shows on the **Map** page
and the governor's profile.

### KvK / event scoring
**KvK DKP** page → pick a saved event (or set a date range) → see points from
kills + deads gained in that window. Admins create events on the back end.

### Who did what (audit)
**Audit** page → filter by action / officer / date → **Export CSV** for records.

### Admin-only tasks (Control page)
- **Officers** panel: add/disable officers, reset passwords, force a password change.
- **Automatic scans** panel: schedule daily scans (UTC).
- Promote to **R5**.
