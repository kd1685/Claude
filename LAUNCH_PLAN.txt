ASCENT TERMINAL — LAUNCH PLAN
==============================

Phase 0 — Soft launch (week 1)
-------------------------------
* Server live, TLS green, Stripe webhooks firing
* 5-10 founder keys issued (free / discounted)
* Founders test terminal, scalper bot, Discord alerts
* Collect feedback — fix any critical bugs

Phase 1 — Public launch (week 2-3)
------------------------------------
* Publish Whop listing (Observer, Operator, Architect tiers)
* Publish Patreon (mirror of Observer tier)
* Announce on X with demo video
* Post in relevant Discord servers / trading communities
* Target: 20 paying members in first month

Phase 2 — Growth (month 2-3)
------------------------------
* Weekly Discord signals / analysis posts
* YouTube short-form content (terminal demo, bot results)
* Affiliate / referral program via Whop
* A/B test landing page copy
* Target: 100 paying members

Phase 3 — Scale (month 4+)
----------------------------
* Mobile companion app (React Native shell wrapping terminal)
* More exchange integrations (Binance, Bybit)
* Premium AI tier (more AI calls / day)
* Institutional / prop desk enquiries

Revenue targets
---------------
Month 1:   20 members @ avg $30  =  $600 MRR
Month 3:  100 members @ avg $35  = $3,500 MRR
Month 6:  300 members @ avg $40  = $12,000 MRR

Key metrics to track
---------------------
* MRR (Stripe dashboard)
* Churn rate (cancelled subscriptions / total)
* DAU on terminal (app logs)
* Bot performance (edge_lab results)
* Discord engagement (posts / reactions)
