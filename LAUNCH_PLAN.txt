 ASCENT TERMINAL — LAUNCH PLAN
════════════════════════════════════════════════════════════════════

This is a plain-English checklist of everything that needs to happen
before the site goes live to paying customers.  Tick each item off
as you complete it.

────────────────────────────────────────────────────────────────────
 PHASE 1 — SERVER (do today, ~30 min)
────────────────────────────────────────────────────────────────────

[ ] 1.1  Upload the code to the VPS
         – unzip ascent_terminal_final.zip
         – copy contents into /root/ascent_terminal/ (overwrite)
         – your .env and data/ are untouched

[ ] 1.2  Rebuild and restart services
         docker compose down && docker compose up -d --build
         curl http://localhost:8000/health   → must return {"status":"ok"}

[ ] 1.3  Check logs for errors
         docker compose logs -f --tail 50

────────────────────────────────────────────────────────────────────
 PHASE 2 — DOMAIN & SSL (do today, ~20 min)
────────────────────────────────────────────────────────────────────

[ ] 2.1  DNS: point your domain's A record to the VPS IP
         (changes propagate in 5-60 min)

[ ] 2.2  Install Nginx (if not present):
         apt install nginx

[ ] 2.3  Paste the Nginx server block from UPDATE_NOTES.md §2 into
         /etc/nginx/sites-available/ascent
         ln -s /etc/nginx/sites-available/ascent /etc/nginx/sites-enabled/
         nginx -t && systemctl reload nginx

[ ] 2.4  Get SSL cert:
         apt install certbot python3-certbot-nginx
         certbot --nginx -d your-domain.com
         certbot renew --dry-run   ← confirm auto-renewal works

[ ] 2.5  Verify HTTPS works:  https://your-domain.com/health

────────────────────────────────────────────────────────────────────
 PHASE 3 — STRIPE (do tomorrow, ~30 min)
────────────────────────────────────────────────────────────────────

[ ] 3.1  Create products & prices in Stripe dashboard
         – "Ascent Terminal Monthly"  (e.g. $29/month)
         – "Ascent Terminal Yearly"   (e.g. $249/year)
         Copy both price IDs.

[ ] 3.2  Add to platform/.env on VPS:
         STRIPE_PRICE_MONTHLY=price_xxxx
         STRIPE_PRICE_YEARLY=price_yyyy

[ ] 3.3  Add webhook endpoint in Stripe dashboard:
         URL: https://your-domain.com/api/stripe/webhook
         Events: checkout.session.completed
                 customer.subscription.updated
                 customer.subscription.deleted
                 invoice.payment_failed
         Copy the signing secret → STRIPE_WEBHOOK_SECRET in .env

[ ] 3.4  Test locally:
         stripe listen --forward-to localhost:8000/api/stripe/webhook
         stripe trigger checkout.session.completed

[ ] 3.5  Set PAYMENTS_LIVE=true in .env, restart services.

────────────────────────────────────────────────────────────────────
 PHASE 4 — SMOKE TEST (after each phase)
────────────────────────────────────────────────────────────────────

[ ] 4.1  Register a new account, verify email arrives
[ ] 4.2  Subscribe (use Stripe test card 4242 4242 4242 4242)
[ ] 4.3  Confirm dashboard unlocks after payment
[ ] 4.4  Check at least one live price updates in the UI
[ ] 4.5  Log in as admin, visit /admin/health — all green
[ ] 4.6  Trigger a test alert email

────────────────────────────────────────────────────────────────────
 PHASE 5 — LAUNCH
────────────────────────────────────────────────────────────────────

[ ] 5.1  Set REFERRAL_ENABLED=true if you want the referral system live
[ ] 5.2  Post launch announcement (Twitter/X, Discord, Reddit, etc.)
[ ] 5.3  Monitor docker compose logs -f for the first hour
[ ] 5.4  Set up a cron to back up data/ascent.db daily:
         0 3 * * * cp /root/ascent_terminal/platform/data/ascent.db \
                      /root/backups/ascent_$(date +\%Y\%m\%d).db
