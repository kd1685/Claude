# ✅ WHAT TO DO NOW — owner runbook (post-upgrade)

You've just received the v3.0 upgrade of Ascent Terminal.
Here's exactly what to do next, in order.

──────────────────────────────────────────────────
 RIGHT NOW  (today, ~15 min)
──────────────────────────────────────────────────

1. Pull the latest code
   git pull
   (or double-click UPDATE.bat if git pull doesn't work alone)

2. Run the organiser
   Double-click  organise.bat
   It will create the platform/ folder and copy all scripts.

3. Open  platform\.env  in Notepad
   Fill in:
     MEXC_API_KEY=...
     MEXC_API_SECRET=...
     TELEGRAM_BOT_TOKEN=...
     TELEGRAM_CHAT_ID=...

   If you don't have a Telegram bot yet:
     → Open Telegram → search @BotFather → /newbot

4. Install dependencies (if not already done)
   Open a terminal in  platform/
   pip install -r requirements.txt

5. Start the scalper in PAPER mode
   Double-click  platform\run_bot.bat
   — or —
   cd platform
   python bots/scalper_bot.py

   You should see Telegram messages within a few minutes.
   They will say [PAPER] — that means no real money.

──────────────────────────────────────────────────
 THIS WEEK
──────────────────────────────────────────────────

□ Let the paper bot run for 24-48 h — check the logs
□ Run edge_lab.py to find the best parameters for your symbol
□ Read GO_LIVE_STEPS.md for the full go-live checklist
□ Read LAUNCH_PLAN.txt for the phased rollout plan

──────────────────────────────────────────────────
 WHEN READY TO GO LIVE
──────────────────────────────────────────────────

□ Set  PAPER_TRADE=false  in  platform\.env
□ Set  POSITION_SIZE_USDT=10  (start small)
□ Restart the bot
□ Watch first 5 trades carefully

──────────────────────────────────────────────────
 USEFUL COMMANDS
──────────────────────────────────────────────────

  Start scalper:       python bots/scalper_bot.py
  Start trend bot:     python brain/mexc_trend_bot.py
  Run research:        python brain/edge_lab.py
  Run backtest:        python brain/swing_backtest.py
  Update everything:   double-click UPDATE.bat
