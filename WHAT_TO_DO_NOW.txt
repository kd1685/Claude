# ✅ WHAT TO DO NOW — owner runbook (post-upgrade)

You just pulled the latest AscentTerminal build.  Here's exactly what
to do next, in order.

────────────────────────────────────────────────────────────────────
STEP 1  ·  Verify your .env
────────────────────────────────────────────────────────────────────

Open .env and confirm these are set:

  ACCESS_KEY=<any strong string you want to use as your password>
  ANTHROPIC_API_KEY=sk-ant-…

Optional but recommended:

  EXECUTE_KEY=<strong string — gates live/paper order execution>
  DEFAULT_EXCHANGE=mexc
  USER_CAP_USD=100

If .env doesn't exist yet:
  cp .env.example .env
  nano .env

────────────────────────────────────────────────────────────────────
STEP 2  ·  Install dependencies
────────────────────────────────────────────────────────────────────

  pip install -r requirements.txt

You only need to do this once (and after future updates that add deps).

────────────────────────────────────────────────────────────────────
STEP 3  ·  Start the server
────────────────────────────────────────────────────────────────────

  uvicorn platform.main:app --reload

Or on Windows, double-click UPDATE.bat (it pulls + restarts in one go).

The server starts on http://localhost:8000 by default.

────────────────────────────────────────────────────────────────────
STEP 4  ·  Open the terminal
────────────────────────────────────────────────────────────────────

Go to http://localhost:8000 (or your domain if hosted).
Enter your ACCESS_KEY.  You should see the full terminal UI.

────────────────────────────────────────────────────────────────────
STEP 5  ·  Quick sanity checks
────────────────────────────────────────────────────────────────────

□ CHART tab — pick an asset, candles load
□ SIGNALS tab — indicator panel populates
□ AI tab — click "Analyse this setup" — Claude responds
□ ALERTS tab — webhook URL shown, no errors
□ BOTS tab — "+ ADD TREND BOT" button visible

If any tab shows an error, check the server console for tracebacks.

────────────────────────────────────────────────────────────────────
STEP 6  ·  (Optional) Run a paper bot
────────────────────────────────────────────────────────────────────

1. Click BOTS → + ADD TREND BOT
2. Set symbol (e.g. BTC/USDT:USDT), timeframe 1d, risk 1%
3. Leave mode as PAPER
4. Click START
5. Check the events log — you should see "waiting for signal" every
   poll cycle.

────────────────────────────────────────────────────────────────────
STEP 7  ·  (Optional) Test a webhook
────────────────────────────────────────────────────────────────────

  curl -X POST http://localhost:8000/api/tv-webhook \
    -H 'Content-Type: application/json' \
    -d '{"action":"ALERT","symbol":"BTCUSDT","price":67000}'

Alert should appear in the ALERTS tab within 1–2 seconds.
