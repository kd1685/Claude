# ✅ WHAT TO DO NOW — owner runbook (post-upgrade)

Everything in the platform is built and tested. The only things left
before the site accepts real money are listed below in order.

────────────────────────────────────────────────
 TODAY (takes ~1 hour)
────────────────────────────────────────────────

1.  Upload the code
    – Unzip ascent_terminal_final.zip on your Windows machine.
    – Use WinSCP (or UPDATE.bat) to copy the contents into
      /root/ascent_terminal/ on the VPS.  Click "overwrite" if asked.
    – Your .env and data/ folder are NOT in the zip; they are safe.

2.  Restart Docker
    SSH into VPS:
        cd /root/ascent_terminal
        docker compose down && docker compose up -d --build
    Check:
        curl http://localhost:8000/health   → {"status":"ok"}

3.  Point your domain
    – DNS: A record → your VPS IP
    – Wait for propagation (5-60 min), then:
        certbot --nginx -d your-domain.com

4.  Verify HTTPS
        curl https://your-domain.com/health  → {"status":"ok"}

────────────────────────────────────────────────
 TOMORROW (takes ~30 min)
────────────────────────────────────────────────

5.  Set up Stripe (see GO_LIVE_STEPS §3 for the exact commands)
    – Create Monthly + Yearly prices in Stripe dashboard.
    – Add price IDs + webhook secret to platform/.env on VPS.
    – Register the webhook endpoint in Stripe.
    – Test with:  stripe trigger checkout.session.completed
    – Set PAYMENTS_LIVE=true in .env, restart Docker.

6.  Do a final smoke test
    – Register → subscribe (test card 4242…) → confirm dashboard unlocks.
    – Visit /admin/health — all green.

────────────────────────────────────────────────
 THEN YOU'RE LIVE — go announce it!
────────────────────────────────────────────────

Optional next steps (whenever you're ready):
  • Set REFERRAL_ENABLED=true to activate the referral system.
  • Add ALERT_WEBHOOK_URL to get Discord/Slack alerts.
  • Set up a daily DB backup cron (see LAUNCH_PLAN.txt §5).
