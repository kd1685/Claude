Put the Cloudflare Origin Certificate here (GO_LIVE_STEPS.md §11):
  origin.pem        the certificate PEM
  origin-key.pem    the private key PEM
Created in Cloudflare: SSL/TLS -> Origin Server -> Create Certificate
(defaults: RSA, yourdomain + *.yourdomain, 15 years).
These are SECRETS - never commit or share them.
