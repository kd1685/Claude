Seeing "Windows protected your PC"?

New apps from small publishers earn Microsoft SmartScreen's trust over time, so early downloads can show this prompt. It doesn't mean anything was detected — it means the file is new. To continue:

  1. Click More info on the blue prompt.
  2. Check the app name reads Ascent Terminal, then click Run anyway.
  3. Optionally verify the file is genuine with the checksum below before running.

Verify your download (SHA-256)

In PowerShell run Get-FileHash AscentTerminal-Setup-1.0.0.exe and confirm it matches:

REPLACE-WITH-REAL-CHECKSUM-AFTER-BUILDING — certutil -hashfile AscentTerminal-Setup-1.0.0.exe SHA256
